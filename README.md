# PackChecker

「ジムに来たのに靴を忘れた」という絶望を未然に防ぐための、目的別持ち物チェッカーアプリです。
ジム用、サウナ用など、行き先に応じたパッキングリストをシンプルかつ直感的に管理することができます。

---

## Overview
PackCheckerは、日常のちょっとした忘れ物をなくすために作られたiOSアプリです。
最新のiOSデザインを取り入れた美しいグラデーションのカードUIと、純正メモアプリのような滑らかな操作感（クイックルックなど）を備えています。

---

## Features
* **目的別のリスト管理**: 「行き先」と「持ち物」をデータとして管理。
* **モダンUI**: 8種類のグラデーションカラーからカードの色を選択可能。自身の好みに合わせて項目をカテゴライズすることができます。
* **クイックルック機能**: リスト画面でカードを長押しするとポップアッププレビューが表示されます。
* **直感的な編集・削除**:ナビゲーションバーの右上のボタンから簡単にリストの再編集や削除・新規追加が可能です。

## Usage

### 1. 行き先の追加

* トップ画面（"Where are you going?"）右上の「＋」ボタンをタップします。

* 行き先の名前（例: ジム）、必要な持ち物、そしてリストのカラーを選択して「保存」を押します。

### 2. 持ち物の確認

* トップ画面に並んだ色鮮やかなカードをタップすると、持ち物一覧画面が開きます。出発前にここを見て荷物をチェックしましょう。

### 3. クイックルック

* トップ画面でカードを長押しすると、画面を切り替えずに中身のアイテム一覧をサッと確認できます。

### 4. 編集・削除

* トップ画面でカードを長押ししてメニューから選択するか、詳細画面右上の「…」をタップすることで、リストの内容を変更したり、不要になったセットを削除できます。
